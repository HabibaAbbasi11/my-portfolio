★ Featured Project 02

GlamFusion

A full-stack desktop fashion simulation application where users apply makeup, hairstyles, and outfits on a base model. Built in Java Swing with a multi-layer PNG rendering engine, connected to a PHP/MySQL backend via a custom HTTP client-server bridge.

  • Multi-layer PNG rendering engine for real-time outfit overlays
  • Close-up Makeup Page + full-body Style Studio viewport
  • Java HTTP client-server bridge → PHP/MySQL profile persistence
  • Modular layouts: GridBagLayout, BorderLayout, JScrollPane
  • 6 screens: Login, Signup, Dashboard, Lookbook, Makeup, Style
  • Custom font integration & purple-pink premium aesthetic
Java Java Swing PHP MySQL XAMPP Eclipse IDE
